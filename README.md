# TurnoSmart V25 — Foto de OEE legível

## Problema corrigido

A versão anterior mostrava a coluna tratada pelo OCR como grandes manchas pretas, dificultando a conferência dos números escritos à mão.

## Novo fluxo

1. O aplicativo recorta a coluna correta.
2. A prévia principal permanece colorida e legível.
3. Uma imagem tratada separada é usada apenas internamente pelo OCR.
4. Cada máquina mostra a miniatura da própria linha ao lado do campo de OEE.
5. O usuário confirma ou corrige o valor antes de analisar.

A leitura automática de escrita manual continua sendo apenas uma sugestão. Nenhum valor precisa ser aceito sem conferência.
